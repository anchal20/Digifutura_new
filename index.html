Healthy Surprise

Healthy Surprise is the fun and easy way to eat healthy!
Healthy snacks subscription site built with Ruby on rails and Twitter bootstrap.
Healthy surprise is a subscription model based web application. It was built with functionality to track the orders, updating the dashboard, switching the products, updating the subscriptions, showing order status and scheduling orders. Site also manages the addresses, deactivating and reactivating the subscriptions and updating delivery dates..
API's used:
- Chargify API: manage subscriptions. This API is used intensively since users can do many things with subscriptions.
- Shipstation API: manage orders status. One subscription has many orders.
- Capacity LLC API: packaging and shipment handler.
- USPS API: tracking boxes and estimation delivery time.
- Elastic search: searching orders/users on admin.
Team size- 2 engineers
Duration- 2 months

The Funnel
